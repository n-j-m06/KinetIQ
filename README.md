# KINETIQ Backend

AI-powered Personalized Driver Behaviour Intelligence System.

## Tech Stack

- FastAPI
- PostgreSQL (Neon)
- SQLAlchemy
- JWT Authentication
- WebSockets
- Pydantic

## Features

- Authentication
- Driver Profiles
- Driving Sessions
- Metrics Engine
- Alerts Engine
- Behaviour Fingerprinting
- Analytics Dashboard
- Real-time WebSockets

## Run Locally

```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
```

Swagger:

http://127.0.0.1:8000/docs
